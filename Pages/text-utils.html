    • Text-Utilities

    Lekhni supports a variety of text styles and all of them are available to reuse.

    Heading

    Lekhni provides four different types of headings. You can import any of these to your project with following classnames : heading-lg. heading-md, heading-sm and heading-sub.

    This is my first heading
    This is my second heading
    This is my third heading
    This is sub heading
    
            <div class="heading-lg">This is my first heading</div>
            <div class="heading-md">This is my second heading</div>
            <div class="heading-sm">This is my third heading</div>
            <div class="heading-sub">This is sub heading</div>
    Text

    We've a variety of texts as shown below. They can be imported with text-sm, text-grey, text-centered and text-capitalized

    This is usual text size text
    This text will be gray
    This text would be centered
    This text would be capitalized
    
          <div class="text-sm">This is usual text size text</div>
          <div class="text-grey">This text will be gray</div>
          <div class="text-centered">This text would be centered</div>
          <div class="text-capitalized">This text would be capitalized</div>
