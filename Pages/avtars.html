    • Avtars

    Avatars are user profile pictures, usually found in circular shapes. Avatars can be used for 'container' objects — projects, spaces, repositories, etcetera — to give them a recognisable visual identity.

    Profile Avtars

    Profile Avatars can be created using the class avtar, inside a div which has the img tag. For different sizes, you can use sm, md or lg on img tag. 

    

            <div class="avtar">
            <img
                class="sm"
                src="../../Assets/imgs/male profile.jpg"
                alt="avtar"
              />
              <img
                class="md"
                src="../../Assets/imgs/male profile.jpg"
                alt="avtar"
              />
              <img
                class="lg"
                src="../../Assets/imgs/male profile.jpg"
                alt="avtar"
              />
              </div>
    Squared Avtars

    Sqaured Avatars can be created using the class square inside the img tag. For different sizes, you can use sm, md or lg on img tag. 

    

          <div class="avtar">
          <img
              class="square sm"
              src="../../Assets/imgs/male profile.jpg"
              alt="avtar"
            />
            <img
              class="square md"
              src="../../Assets/imgs/male profile.jpg"
              alt="avtar"
            />
            <img
              class="square lg"
              src="../../Assets/imgs/male profile.jpg"
              alt="avtar"
            />
            </div>
