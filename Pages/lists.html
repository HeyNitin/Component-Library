    List for use for listing elements example contact list. Spaced list or stacked list.

    Spaced lists

    To use the spaced list, simply use classname list

  • Home
  • Blogs
  • About us 
    • 
            <div class="list">
            <li>
            <a>Home</a>
            </li>
            <li>
            <a>Blogs</a>
            </li>
            <li>
            <a>About us</a>
            </li>
            </div>
    Stacked lists

    To use the stacked list, simply use classname list-stacked. Also, you can use alternate to give a list item different color

  • Home
  • Blogs
  • About us 
    • 
          <div class="list-stacked">
          <li>
          <a>Home</a>
          </li>
          <li class="alternate">
          <a>Blogs</a>
          </li>
          <li>
          <a>About us</a>
          </li>
          </div>
