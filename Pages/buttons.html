    Buttons are the soul of an interactive Web Application. Buttons allow users to take actions, and make choices, with a single tap. Use class button to use a simple button with no text decoration or choose from the below styles to go with it.

    Primary Button

    Primary buttons are high-emphasis, distinguished by their use of elevation and fill. They contain actions that are primary to your app. Use class btn-primary to use this style. 

    
            <buttonclass="button btn-primary">
            <a>Click Me</a>
            </button>
    Link Button

    link buttons have the same styles but they can contain a link on the text as decoration is set to none. Use class btn-link to use this style. 

    
            <button class="button btn-link">
            <a href = "URL here">Click Me</a>
            </button>
    Icon Button

    These support an icon inside the button. I've use google icons for this you can choose any icon of your desire. Use class btn-icon to use this style. 

    
            <button class="button btn-icon">
            <i class="material-icons">save</i>
            <a>Click Me</a>
            </button>
    Floating Button

    Floating buttons are used for many purposes. They remain on top layer of the screen. Use class btn-floating to use this style. 

    
          <button class="button btn-floating">
          <a class="material-icons">shop</a>
          </button>
