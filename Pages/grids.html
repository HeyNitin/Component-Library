    • Grids

    Grids are used for layouting the elements in row and column manner. Note: grid-item property will give some default style with border and background, remove that class if not needed.

    Two Column Grid

    To use two column grids use classname as grid-two

    First column
    Second column
    
            <div class="grid-two">
            <div>First column</div>
            <div>Second column</div>
            </div>
    Three Column Grid

    To use three column grids use classname as grid-three

    First column
    Second column
    Second column
    
          <div class="grid-three">
          <div>First column</div>
          <div>Second column</div>
          <div>Third column</div>
          </div>
