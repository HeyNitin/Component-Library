    • Round Images

    Round Images are the circular, with a default dimension of 160px X 160px. Use this with classname as img-round in img tag. 

    
            <div>
            <img
                class="img-round"
                src="URL here"
                alt="img"
              />
              </div>
    Responsive Images

    Full Width responsive Image is the one that expands to cover the container it is present in. Resize the browser window, to see the responsive property. Use this with classname as img-responsive in img tag. 


    
          <div>
          <img
              class="img-responsive"
              src="URL here"
              alt="img"
            />
            </div>
