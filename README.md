# Lekhni-UI 

## Description- 
Lekhni is an easy to use, customizable CSS Framework, helping people to create their Dream Projects Better and Faster. 

## Technologies Used - 
- HTML
- Vanilla CSS
- JavaScript

## Components Defined in Lekhni - 
- Avatar
- Alert
- Badge
- Button
- Card
- Grid
- Image
- Input
- List
- Modal
- Navigation
- Ratings
- Text-Utilities
- Snack-Bar
